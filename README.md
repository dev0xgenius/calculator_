# Calculator App

A themed calculator application with three color schemes and responsive design. Built with vanilla JavaScript, CSS custom properties, and mobile-first approach.

[View Live Demo](https://0xgenius-calculatorv2.netlify.app) | [Frontend Mentor](https://www.frontendmentor.io/profile/dev0xgenius)

---

## Features

- Basic arithmetic: addition, subtraction, multiplication, division
- Three color themes (toggle via radio buttons)
- Responsive layout for all screen sizes
- Keyboard-style button grid
- Delete and reset functionality
- Theme persistence inspiration from prefers-color-scheme

## Tech Stack

- **Language:** Vanilla JavaScript, HTML5, CSS3
- **Styling:** CSS custom properties, Flexbox, CSS Grid
- **Theming:** Dynamic stylesheet switching
- **Build:** None (vanilla project)

## Getting Started

```bash
git clone https://github.com/dev0xgenius/calculator_.git
cd calculator_
# Open index.html in browser or use live-server
```

## Themes

| Theme | Style | Description |
|-------|-------|-------------|
| Theme 1 | Default | Dark calculator theme |
| Theme 2 | Light | Clean light theme |
| Theme 3 | Purple | High-contrast purple theme |

## Project Structure

```
css/
  themes/
    theme-1.css     # Default dark theme
    theme-2.css     # Light theme
    theme-3.css     # Purple theme
  base.css          # Base styling
app.js              # Calculator logic
theme.js            # Theme switching
index.html          # Entry point
```
